# Default Circuit Analysis

## Circuit: "MyCircuit"

This is the default example circuit generated by the NeuromorphicWizard.

## Components

| Group | Plasmid | Amount | Role |
|-------|---------|--------|------|
| X1 | Csy4 | 150 ng | ERN enzyme (cuts Csy4-recognized mRNA) |
| X1 | mKO2 | 100 ng | Orange fluorescent reporter (unregulated) |
| X2 | Csy4_rec_CasE | 100 ng | CasE, but its mRNA is cut by Csy4 |
| X2 | eBFP2 | 100 ng | Blue fluorescent reporter (unregulated) |
| Bias | CasE_rec_mNeonGreen | 200 ng | mNeonGreen, but its mRNA is cut by CasE |

## Circuit Logic

```
Csy4 ──┐
       ├── (co-transfected with mKO2, orange reporter)
       │
       ▼ inhibits
   CasE (from Csy4_rec_CasE)
       │
       ├── (co-transfected with eBFP2, blue reporter)
       │
       ▼ inhibits
  mNeonGreen (from CasE_rec_mNeonGreen)
```

### Step by step:

1. **Csy4 is produced** (150 ng, high expression) -- this is an endoribonuclease
2. **Csy4 cuts CasE mRNA** -- because `Csy4_rec_CasE` means CasE's mRNA has a Csy4 recognition site. So CasE protein production is **suppressed**.
3. **CasE cannot cut mNeonGreen mRNA** -- because CasE is suppressed by Csy4. So `CasE_rec_mNeonGreen` is effectively just mNeonGreen with no one cutting it.
4. **mNeonGreen glows green** -- double negative: Csy4 inhibits the inhibitor (CasE) of mNeonGreen.

### Reporters:

- **mKO2 (orange)**: Always on (unregulated, in X1 group). Confirms transfection worked.
- **eBFP2 (blue)**: Always on (unregulated, in X2 group). Confirms transfection worked.
- **mNeonGreen (green)**: Should be ON because its inhibitor (CasE) is itself inhibited by Csy4.

## Expected Outcome

All three fluorescent proteins should be visible:
- Orange (mKO2) -- constitutive control
- Blue (eBFP2) -- constitutive control
- Green (mNeonGreen) -- the circuit output, ON due to double inhibition
